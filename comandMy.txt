4. процедура отправки в гит хаб

4.1. git init

4.2. git status

4.3. git add . / git add --all .

4.4. git commit -m "коментарий ваши к обновлению или же название"

4.4.1. привер: git commit -m "test"

4.5. git remote add origin "ваше ссылка" в конце ссылки надо вставить ".git"

4.5.1. пример: git remote add origin https://github.com/armanarstambekov/my-first-blog.git

4.6. git push -u origin HEAD / git push -u origin master
